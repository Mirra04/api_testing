<h2>🔗 API-тестирование в Postman</h2>

В рамках учебного проекта было проведено тестирование REST и SOAP API.

<h3>REST API</h3>

Для веб-приложения DemoShopping была создана коллекция Postman на основе Swagger-документации.

Для запросов реализованы автоматические проверки с использованием Postman Scripts:

<ul>
    <li>Проверка статус-кодов</li>
    <li>Проверка содержимого ответов</li>
    <li>Проверка времени ответа</li>
    <li>Сохранение данных в переменные коллекции для последующих запросов</li>
</ul>

    <li>📄 <a href="https://intern.demoshopping.ru/api-docs/">Swagger API</a></li>
</ul>

<h3>SOAP API</h3>

На основе WSDL-сервиса была создана коллекция Postman для работы с SOAP-запросами.

Коллекция позволяет получать:

<ul>
    <li>Информацию о стране</li>
    <li>Информацию о валюте страны</li>
    <li>Информацию о языке страны</li>
</ul>
