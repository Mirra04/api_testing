<h2>🔗 API-тестирование</h2>

В рамках учебного проекта было проведено тестирование REST и SOAP API.

<h3>REST API</h3>

Для веб-приложения DemoShopping была создана коллекция Postman на основе Swagger-документации. Были протестированы основные методы API, связанные с авторизацией пользователей, каталогом товаров и корзиной.

    <li>📄 <a href="https://intern.demoshopping.ru/api-docs/">Swagger API</a></li>
</ul>

<h3>SOAP API</h3>

На основе WSDL-сервиса была создана коллекция Postman для работы с SOAP-запросами.

Коллекция позволяет получать:

<ul>
    <li>Информацию о стране</li>
    <li>Информацию о валюте страны</li>
    <li>Информацию о языке страны</li>
</ul>
